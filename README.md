## Setup

```sh
git clone https://github.com/hwp-labs/zentry.git

cd zentry

npm install

npm run dev

http://localhost:3000
```

## Documentation

|   # | Feature      | Summary                                                                                                                    |
| --: | :----------- | :------------------------------------------------------------------------------------------------------------------------- |
|   1 | Pastors      | Bio-data of pastors and their respective branches where applicable                                                         |
|   2 | Workers      | Bio-data of church workers and their respective departments                                                                |
|   3 | Members      | Bio-data of all church members                                                                                             |
|   4 | Houses       | List of house groups, names and contact info of their respective leaders and members                                       |
|   5 | Departments  | List of church departments, names and contact info of their respective HODs and workers                                    |
|   6 | Finances     | Revenue and expense records of the church such as tithes and offerings, and other financial ledgers                        |
|   7 | Birthdays    | List of upcoming birthdays and baby dedication with push notifications                                                     |
|   8 | Weddings     | List of upcoming weddings and wedding anniversaries with push notifications                                                |
|   9 | #Connect     | Online business directory of all church members                                                                            |
|  10 | Events       | Details of upcoming events on the church calendar with push notifications                                                  |
|  11 | Data Capture | Quick access dual-functionality button for bio-data capture and financial records capture - based on Operator account type |
|  12 | Resources    | Download church resource materials, messages, publications, books, music, videos, etc.                                     |
